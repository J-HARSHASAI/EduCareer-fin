MTech in Computer Science & Engineering

Master advanced concepts in computing and technology

Course Description

The MTech in Computer Science & Engineering is designed for students aiming to excel in cutting-edge areas like artificial intelligence, machine learning, cloud computing, and big data analytics. This program provides in-depth knowledge of advanced algorithms, system architecture, and software development, preparing graduates for high-impact roles in technology and research.

Relevant Subjects

  • Advanced Algorithms and Data Structures
  • Machine Learning and Deep Learning
  • Artificial Intelligence and Expert Systems
  • Cloud Computing and Distributed Systems
  • Big Data Analytics and Data Mining
  • Computer Networks and Security
  • Software Engineering and DevOps

Required Skills

  • Strong programming skills in languages like Python, Java, or C++
  • Proficiency in data structures and algorithms
  • Understanding of operating systems and computer architecture
  • Analytical and problem-solving abilities
  • Familiarity with machine learning frameworks (e.g., TensorFlow, PyTorch)
  • Knowledge of database management and SQL
  • Ability to work in collaborative, agile environments
